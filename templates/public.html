Seguridad del Paciente

Detección inmediata de escape de zona

El GPS PG12 se lleva oculto en la ropa del paciente. Flask consulta la API cloud del dispositivo cada 30–60 segundos y evalúa con PostGIS si la ubicación sigue dentro de alguna zona segura. Si no — la alerta se dispara de inmediato, con trazabilidad completa hasta el evento GPS.

  • GPS PG12 oculto en ropa — el paciente no interactúa con él
  • Lecturas almacenadas en MongoDB por alto volumen de escritura
  • Alerta instantánea en PostgreSQL si sale de todas las zonas seguras
  • Escalación a contacto de emergencia si no se atiende en 5 min
  • Trayectoria y última posición visibles en el dashboard de alertas

Rondas de Cuidadores

Trazabilidad de presencia
en cada zona del centro.

Los beacons FeasyBeacon están fijos en puntos estratégicos del edificio — puertas, salidas, sala común, comedor. El teléfono Android del cuidador los detecta al pasar mediante Web Bluetooth y registra automáticamente la ronda.

No se requieren gateways BLE adicionales. El teléfono del cuidador es el receptor. Cada detección queda almacenada en MongoDB con RSSI, zona, cuidador y timestamp.

Los beacons no se asignan a pacientes — son puntos de referencia fijos del edificio que permiten saber qué zonas fueron visitadas y cuándo.

Pulsera NFC

Verificación de medicación
con un solo toque.

El paciente lleva una pulsera NFC pasiva (sin batería, como una pulsera hospitalaria). El cuidador acerca su teléfono Android para confirmar identidad y registrar la administración del medicamento.

1

Paciente porta la pulsera NFC

Tag NFC DESFire pasivo — sin batería, sin interacción del paciente. Contiene el ID de receta grabado en memoria NDEF. Se porta como pulsera de identificación hospitalaria.

2

Cuidador acerca su teléfono

Chrome Android lee el tag vía Web NFC API (requiere HTTPS). La webapp confirma el ID de receta, el paciente y el medicamento antes de registrar la administración.

3

Evento registrado en PostgreSQL

Se inserta en lecturas_nfc con FK a recetas y dispositivos. Alimenta el indicador de adherencia terapéutica del paciente. Bajo volumen, alta integridad clínica.
